💛 AI Emotional Support Chatbot (MindCare)

An intelligent, human-like emotional support chatbot designed to help users express their feelings, reduce stress, and improve mental well-being.

This application provides a safe and supportive space where users can talk freely and receive thoughtful, context-aware responses powered by AI.

---

✨ Features:

* 🧠 Emotion-aware AI responses (human-like conversation)
* 😊 Mood selection system (Happy, Sad, Angry, Tired, etc.)
* 💬 Smart chatbot with dynamic replies (no repetition)
* 📊 Mood tracking dashboard with weekly analytics
* 🔥 Daily streak system for user engagement
* 🌙 Light / Dark mode support
* 🎙️ Voice input support (optional)
* 🗂️ Chat history with rename & delete options
* 💛 Clean, modern, premium UI

---

⚙️ Tech Stack:

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* AI: OpenAI API
* Charts: Chart.js

---

🎯 Purpose:

Many people struggle with stress, loneliness, and emotional pressure but don’t always have someone to talk to.

This project aims to bridge that gap by providing:

* Instant emotional support
* A judgment-free space
* A companion-like experience

---

🚀 Future Improvements:

* Memory-based conversations
* Emotion detection (face + text)
* Personalized suggestions
* Real-time notifications
* Mobile app version

---

💡 Inspired by:
Mental Health Awareness (SDG 3 – Good Health & Well-being)
